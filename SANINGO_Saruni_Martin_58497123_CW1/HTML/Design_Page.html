For CW3, The Following JavaScript ES6 Features and functions from the cookbook were used:


The use of 'let'

I used 'let' because it is block-scoped, meaning it is limited in scope to the block (e.g., inside a loop or an if statement) where it is defined. This helps prevent variable hoisting issues and makes the code more predictable and maintainable.
Used in Apply_page script lines 34-37, 87-91, 138-141, 186-189,...

The use of 'set'

I used the set property because the array I formed 'emptyCellRows' had repeating numbers and because a set can only take an item once, it was the most ideal way for me to remove duplicates and form a new array 'uniqueemptyCellRows' without duplicates
Used in Apply_page script line 653

The use of 'class'

In hiding and displaying the majors in each college in the Apply_page, I used JavaScript to create the class show to help display/hide the elements
Used in Apply_page line 15-17, 21-23, 26-28
